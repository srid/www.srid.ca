Use Neuron, get RSS for free

Atom, not RSS - but you get the idea.

Instead of writing your own static site generator, and write code to laboriously generate RSS feeds (which are all useful exercise in learning) - why not use Neuron 1 , and add the following to your, say, blog.md and call it a day? 

---
feed:
  count: 5
---

That is all it takes to generate an atom feed, blog.xml, that will aggregate all of your blog posts linked from blog.md. See neuron‚Äôs documentation on the Web feeds plugin for details.
